1. Regisztráció: Bejelentkezés gombra kattintunk -> Az e-mail és jelszó mezőbe még nem létező profilunk címét és jelszavát próbáljuk beírni: ujfelhasznalo@gmail.com, UjFelhasznalo123; A nem létező autentikációs adatokra "Váratlan hiba történt" visszajelzést várunk -> Regisztráció gombra kattintva megadjuk a mezőkben új profilunk adatait: Név: UjFelhasznalo, E-mail: ujfelhasznalo@gmail.com, Jelszó: UjFelhasznalo123; Ekkor azt várjuk, hogy a felület az új profillal beléptessen, a belépett felhasználó helyességét az UjFelhasznalo felhasználónév jelzi a felületen -> Kijelentkezés gombbal kilépünk -> Bejelentkezés gombbal ismét megkíséreljük a belépést a már létező új profilunkkal a korábbi belépési adatokkal: E-mail: ujfelhasznalo@gmail.com, Jelszó: UjFelhasznalo123; Siker esetén hiba helyett mostmár beléptet a felületre és ismét az UjFelhasznalo név jelenik meg mint profilnév

2. User bejelentkezés: Bejelentkezés gombra kattintunk -> Az e-mail és jelszó mezőbe az alapfelhasználó adatait visszük be: user@user.com, USER_user; Siker esetén a bejelentkezett felületre kerülünk, ahol a felhasználónév user, amennyiben jó fiókot léptetett be a rendszer

3. Termékek kosárba: 2-es folyamat (lásd: 2. User bejelentkezés) -> A bejelentkezett oldalon a Termékek gombra kattintunk -> Egy Sajt terméket adunk a kosárhoz az Add To Cart gombra kattintva -> Az Élelmiszer gombra kattintva azt várjuk, hogy csak élelmiszerek legyenek felsorolva a terméklistában -> Újabb Sajt terméket adunk a kosárhoz az Add To Cart gombra kattintva -> Az Elektronika gombra kattintva csak elektronikai termékeket szabad látnunk -> Az árbeállító csúszkán a maximum értéket 79000-re állítjuk, az elvárások szerint ekkor csak olyan termékek maradhatnak a listán, amik ennél nem kerülnek többe -> Egy Alaplap terméket adunk a kosárhoz az Add To Cart gombra kattintva -> A Sport gombra kattintunk -> Az árat most 4000 és 95000 közé állítjuk a csúszkán, azt várva, hogy az olcsóbb és drágább termékek eltűnjenek a listáról -> Egy Roller terméket adunk a kosárhoz az Add To Cart gombra kattintva -> A Kosár gombra kattintunk, elvárásaink szerint 2 sajt, 1 alaplap, és 1 roller terméknek kell megjelennie a kosárban, mivel ezeket (és ennyit) adtunk hozzá a kosárhoz -> Az Alaplapot kivesszük a kosárból a hozzá tartozó Remove gombra kattintva, azt várjuk, hogy így 2 sajt és 1 roller maradjon csak a kosárban

4. Megrendelés: 3-as folyamat (lásd: 3. Termékek kosárba) -> A megtöltött kosarunk után a Megrendel gombra kattintunk, ami után siker esetén a Megrendelések oldalra kerülünk, ahol a Sajt és Roller termékek lesznek felsorolva a termékeknél Unread státusszal -> Visszamegyünk a Kosárra,aminek már üresnek kell lennie, mivel a termékek meg lettek rendelve-> a Megrendelések gombra kattintunk, azon belül pedig a Számla gombra, ami letölti a számlánkat PDF formátumban

5. Admin bejelentkezés: Bejelentkezés gombra kattintunk -> Az e-mail és jelszó mezőbe a rendszergazdai felhasználó adatait visszük be: admin@admin.com, ADMIN_admin; Siker esetén a bejelentkezett felületre kerülünk, ahol a felhasználónév admin, amennyiben jó fiókot léptetett be a rendszer

6. Megrendelés státusz: 4-es folyamat (lásd: 4. Megrendelés) -> a user profillal való rendelés után a Kijelentkezés gombbal kilépünk -> 5-ös folyamat (lásd: 5. Admin bejelentkezés) -> Immár rendszergazdai profillal a Megrendelések gombra kattintunk -> a Számlát Processing státuszúra állítjuk, amitől egyből meg is kell változnia a státusznak -> Kijelentkezünk a profilból -> 2-es folyamat(lásd: 2. User bejelentkezés) -> ismét a user profillal a Megrendelések gombra kattintunk, ahol azt várjuk, hogy szintén megváltozott a státusz Processing-re
